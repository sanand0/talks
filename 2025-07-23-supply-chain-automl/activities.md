# Activities

1. **Give an LLM agent an IoT dataset and an email-style analytics brief**
  - Who: speaker
  - When: during the talk
  - Model/tool: Strive Intelligence tool, LLM agent
  - What happened: Anand uploaded the IoT data and asked the agent to find the top drivers of low throughput and yield loss.
  - Source:
    > `We took our application, uploaded the data set.`
    > `I'm going to actually take this, literally this email, and put it into the Strive Intelligence tool, which is an agent.`

2. **Use an LLM agent with code execution to analyze fleet data**
  - Who: speaker
  - When: during the talk
  - Model/tool: Strive Intelligence tool, ChatGPT-like agent
  - What happened: The agent wrote code, ran analysis, corrected itself, and returned recommendations.
  - Source:
    > `It has access to a container, meaning it has access to code`
    > `it has come up with a set of conclusions`

3. **Build a webcam-safety web app with Bedrock Claude 3.5 Sonnet**
  - Who: speaker
  - When: during the talk
  - Model/tool: Bedrock Claude 3.5 Sonnet
  - What happened: Anand asked an agent to create an app that sends webcam images to an LLM and flags safety risks.
  - Source:
    > `build a web application that lets the user take a picture from their webcam, send it to an LLM`
    > `I'm sending this to Bedrock's Claude 3.5 Sonnet model.`

4. **Chain business analyst, coder, and deployer agents**
  - Who: speaker
  - When: during the talk
  - Model/tool: business analyst agent, coding agent, deployment agent
  - What happened: The prompt was translated, coded, and deployed by separate AI agents.
  - Source:
    > `The first agent is someone who understands the requirements`
    > `the third agent ... is a deployer agent`

5. **Have LLMs write code, run it, and loop on failures**
  - Who: audience
  - When: during the talk
  - Model/tool: LLMs, agentic loop
  - What happened: Anand urged attendees to try the write-run-revise pattern on their own problems.
  - Source:
    > `Have LLMs write code, run them, and put them in a loop`
