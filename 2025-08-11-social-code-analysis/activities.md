# Activities

1. **Have students write a Python script that uses an LLM to analyze and narrate data**
  - Who: audience
  - When: before the talk
  - Model/tool: LLM
  - What happened: Students were assigned to build automated analysis scripts that create a story from a dataset.
  - Source:
    > `Write a Python script that uses an LLM to analyze, visualize, and narrate a story from a dataset.`

2. **Have students prove their AI output is high quality**
  - Who: audience
  - When: before the talk
  - Model/tool: LLM
  - What happened: Students had to convince an LLM that their script and output were good enough.
  - Source:
    > `Convince an LLM that your script and output are of high quality.`

3. **Use an LLM to evaluate only carefully chosen analysis**
  - Who: speaker
  - When: before the talk
  - Model/tool: LLM
  - What happened: The evaluation rubric checked that only relevant analysis was sent to the model.
  - Source:
    > `the code should only send carefully chosen analysis to the LLMs, not the whole thing.`
    > `The code only processes relevant analyses by summarizing key findings`

4. **Allow generative AI tools in the course**
  - Who: audience
  - When: during the talk
  - Model/tool: generative AI tools
  - What happened: Anand confirmed that students could use generative AI tools in the course.
  - Source:
    > `So they can also use Generative AI tools?`
    > `Yeah. So in this course it's allowed.`
