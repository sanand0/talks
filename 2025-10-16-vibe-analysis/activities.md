# Activities

1. **Ask the audience to bring personal or synthetic data**
  - Who: audience
  - When: before the talk
  - Model/tool: ChatGPT, Gemini, Claude, LLMs
  - What happened: Participants were asked to download data sources they could analyze.
  - Source:
    > `Let's pick some personal and synthetic datasets`
    >
    > `Download any of these`

2. **Upload slide snapshots into Perplexity for instant explanation**
  - Who: speaker
  - When: before the talk
  - Model/tool: Perplexity
  - What happened: Nirmal used Perplexity to explain other conference slides like a 5-year-old.
  - Source:
    > `I uploaded every one of those slides into Perplexity`
    >
    > `"ELI5." ELI5 is "Explain Like I Am a 5-year-old."`

3. **Use AI to synthesize analysis from WhatsApp exports**
  - Who: speaker
  - When: during the talk
  - Model/tool: Claude
  - What happened: Anand gave a WhatsApp export to Claude and asked for analysis.
  - Source:
    > `give it the data and tell it to analyze`
    >
    > `I'm going to have Claude run this analysis as well`

4. **Ask AI for creative analysis ideas before running code**
  - Who: speaker
  - When: during the talk
  - Model/tool: ChatGPT, Claude
  - What happened: The workshop separates planning from execution and asks for interesting analyses first.
  - Source:
    > `What interesting, useful analysis can I run on this dataset?`
    >
    > `find the most interesting, useful, and non-obvious insights`

5. **Use AI to generate a visual data story**
  - Who: speaker
  - When: during the talk
  - Model/tool: LLM / Codex
  - What happened: The speaker asked for a beautiful visual story in the style of the New York Times.
  - Source:
    > `Create a visual data story as a single-page web app`
    >
    > `in the style of the New York Times`

6. **Publish the generated story to GitHub Pages**
  - Who: speaker
  - When: during the talk
  - Model/tool: GitHub Pages after LLM-generated files
  - What happened: The AI-generated script, styles, and HTML were uploaded and deployed.
  - Source:
    > `create an analysis.py file that will recreate this analysis`
    >
    > `you have Vibe analyzed, Vibe coded, Vibe deployed`
