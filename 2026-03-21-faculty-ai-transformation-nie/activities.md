# Activities

1. **Use ChatGPT and Claude Code to design AI-era testing topics and assessment questions**
  - Who: speaker
  - When: during the talk
  - Model/tool: ChatGPT, Claude Code
  - What happened: Anand asked AI to research what should be taught in an AI-era testing module and then to generate assessment questions from his own notes and prior material.
  - Source:
    > `I was looking at how do we test in the AI era`
    >
    > `I told Codex, and then Claude Code, to analyze and then visualize the student feedback data`
    >
    > `suggest specific questions that align with these ideas`

2. **Generate synthetic attendance data for the Office of Institutional Advancement**
  - Who: speaker
  - When: during the talk
  - Model/tool: ChatGPT
  - What happened: Anand used ChatGPT to produce a realistic synthetic attendance dataset for a live demo.
  - Source:
    > `we can generate synthetic data for attendance`
    >
    > `give me a rich, synthetic data set with the kinds of hypotheses that you would normally find in such data`

3. **Use Codex and Claude Code to analyze and visualize student feedback into a website**
  - Who: speaker
  - When: during the talk
  - Model/tool: Codex, Claude Code
  - What happened: Anand sent student-feedback data to coding agents and got back a mini website summarizing the findings.
  - Source:
    > `I told Codex, and then Claude Code, to analyze and then visualize the student feedback data.`
    >
    > `It built a full-fledged website, effectively a mini presentation of sorts`

4. **Build a catalog of AI-in-education use cases from GitHub, course content, Google Drive, and transcripts**
  - Who: speaker
  - When: during the talk
  - Model/tool: AI / prompt-generated page
  - What happened: Anand asked AI to create a page listing his AI-in-education use cases and to rank them by importance for curriculum redesign.
  - Source:
    > `I gave it my GitHub repository, I gave it my course content, I gave it my Google Drive and all of that`
    >
    > `You create a page that lists all of this.`

5. **Use AI to create cheating-resistant, student-specific questions and collaboration exercises**
  - Who: speaker and audience
  - When: during the talk
  - Model/tool: LLMs
  - What happened: Anand described making different prompts for each student and using secret-agent style collaboration tasks to force human-to-human interaction.
  - Source:
    > `The number one lever I think for AI is every student can get a different question.`
    >
    > `each student is given a secret agent code`
    >
    > `in the age of AI, if you are not using human relationships, you have a problem.`
