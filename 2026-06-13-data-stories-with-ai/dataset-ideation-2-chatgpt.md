## ChatGPT

The best place I found is **Dataful’s Vahan datasets**, not the official Vahan dashboard directly.

For the **largest practical all-India free-style download**, use:

**Dataful: “Master Data: All India level Vehicle Maker and Fuel Norms-wise Total Number of Vehicles Registered in each month”**. It is available as **CSV / XLSX / Parquet**, has **250,861 rows**, **9 columns**, covers **2014–2026**, is sourced from **Parivahan / Vahan**, and is updated monthly. Open the dataset page and use the **CSV / XLSX / Parquet** download option. ([Dataful][1])

This is probably the largest useful single Vahan dataset for analysis because it gives monthly registrations by **maker × fuel norm** across 2014–2026. But for your workshop, I’d be cautious: maker names make the file wider/heavier, and it may be more detailed than beginners need.

For your original “single file under 10 MB, easy in workshop” constraint, I would instead use:

**Dataful: “State, Vehicle Category and Fuel Type-wise Total Number of Vehicles Registered in Each Year”**. It has **22,064 rows**, **8 columns**, covers **2014–2026**, and includes `year`, `state`, `vehicle_category`, `fuel`, and `value`. This is much better for stories like EV adoption, diesel decline, two-wheeler vs car trends, state differences, and consumer spending proxies. ([Dataful][2])

The official **Vahan dashboard** is the source, and it supports dimensions like Vehicle Category, Vehicle Class, Norms, Fuel, Maker, State, Financial Year, Calendar Year, and Month Wise, but it is a dashboard rather than the cleanest one-click dataset source for a workshop. ([Vahan][3])

Use **Parquet** if available; it is likely to stay comfortably smaller and load faster than CSV.
